Step 1 — Fix the Java Problem

Chromis POS requires exactly Java 11. Newer versions cause the startup crash.

⛔ The Error You Are Seeing
Chromis POS Error
An error has occurred during startup:
java.lang.ExceptionInInitializerError
→ at uk.chromis.pos.forms.ChromisFonts.setFont(...)
→ at uk.chromis.pos.forms.StartPOS.main(...)

This error means your installed Java version is incompatible (too recent) with Chromis POS.

1
🗑️ Uninstall your current Java version
▼
  1. Press ⊞ Windows + R at the same time on your keyboard
  2. A small window appears → type appwiz.cpl → press Enter
  3. In the program list, find all entries labelled Java or JDK
  4. Right-click → Uninstall → confirm

Step 2 — Install MariaDB

The database that will store all your sales, products and customers.

💡 What is MariaDB?

Think of MariaDB as a super-fast digital filing cabinet. Chromis POS uses it to save all your products, sales, and customer data. It runs silently in the background — you won't notice it day-to-day, but it's essential.

1
⬇️ Download MariaDB
▼

Go to the official MariaDB website:

🌐 Website Address
https://mariadb.org/download/

Select these options:

  • ✅MariaDB Server (latest stable version)
  • ✅Windows as the operating system
  • ✅File type: .msi (64-bit)
📥
The file is around 70–100 MB. The download may take a few minutes depending on your internet speed.
2

Common Problems & Solutions

If something goes wrong, check these quick fixes.

!
❌ Chromis won't start — Java error
▼

Cause: Java 11 is not recognized, or multiple Java versions coexist.

Fix: Open a Command Prompt and type:

Command Prompt (Administrator)
C:\> java -version
openjdk version "11.0.x" ← must show 11

If it doesn't show Java 11, redo Step 1 and uninstall all Java versions via Programs and Features.

!
❌ Chromis can't connect to MariaDB
▼
  • ⚠️Check that the MariaDB service is running: ⊞ Windows → type Services → find MySQL → must show "Running"
  • ⚠️Make sure the database name is exactly chromispos (all lowercase)
  • ⚠️Double-check your root password (watch out for uppercase letters!)
  • ⚠️The port must be 3306
!
❌ I forgot my MariaDB password
▼

If you have forgotten your MariaDB root password, the simplest fix is to uninstall and reinstall MariaDB, choosing a new password during setup.

⚠️
If you already have data in Chromis, back up the chromispos database from HeidiSQL first before reinstalling!
